# Web open Markdown file links in the preferred editor

Primary activation of a Markdown file link opens the file in the configured preferred editor. The context menu retains integrated preview and relative/full path copy actions.

External web links continue to use upstream's `externalLinkContextMenu` implementation, including Copy Link; no duplicate action module is introduced.

## Validation Coverage

Browser coverage verifies preferred-editor primary activation and preserves preview/copy context actions.
